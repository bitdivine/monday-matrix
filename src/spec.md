# Monday Blues - Simple Web Application

## Overview
A minimalist web application that displays a simple "Hello World" message with clean, centered styling.

## Application Title
Monday Blues

## Core Functionality
- Display the text "Hello World" centered on the screen
- Apply basic styling for clear visibility and aesthetic balance
- Use English as the content language

## User Interface
- Single page application with minimal design
- Text should be prominently displayed in the center of the viewport
- Clean, readable typography with appropriate contrast
- Responsive design that works across different screen sizes

## Technical Requirements
- No backend data storage required
- No user authentication needed
- Static content display only
- Frontend-only application
